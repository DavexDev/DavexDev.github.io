Torre de Hanoi — Demo JS

¿Cómo funciona?

El botón Resolver (auto) genera la secuencia óptima de movimientos con el algoritmo recursivo clásico: mover n−1 discos al auxiliar, mover el disco grande al destino y luego mover los n−1 desde el auxiliar. 

// Esqueleto del algoritmo:
function hanoi(n, from, to, aux, moves) {
  if (n === 0) return;
  hanoi(n-1, from, aux, to, moves);
  moves.push([from, to]);
  hanoi(n-1, aux, to, from, moves);
}
